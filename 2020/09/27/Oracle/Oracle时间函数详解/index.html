Oracle时间函数详解
发表于 2020-09-27| 更新于 2020-10-16|Oracle
阅读量:
函数名 函数详解
current_timestamp 返回当前会话时区中的当前日期:以timestamp with time zone数据类型
current_date 返回当前会话时区中的当前日期
SYSDATE 函数可以得到目前系统的时间
localtimestamp 返回会话中的日期和时间
dbtimezone 返回时区
SESSIONTIMEZONE 返回会话时区
systimestamp

测试：

  • current_timestamp
  • current_date
  • sysdate
  • localtimestamp
  • dbtimezone
  • sessiontimezone
  • systimestamp
