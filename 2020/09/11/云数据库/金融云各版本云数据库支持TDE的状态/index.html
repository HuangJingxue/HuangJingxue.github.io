金融云各版本云数据库支持TDE的状态
发表于 2020-09-11| 更新于 2020-09-11|云数据库
阅读量:
序号 存储引擎 数据库版本 实例类型 支持TDE的状态 备注
1 MySQL 5.6 Primary 支持 RDS for MySQL是和公共云没有区别的 RDS MySQL 8.0高可用版（本地盘）RDS MySQL 5.7高可用版（本地盘）RDS MySQL 5.6
2 MySQL 5.6 Readonly 支持
3 MySQL 5.7 Primary 支持
4 MySQL 8.0 Primary 支持
4 PostgreSQL 9.4 Primary 不支持 RDS PostgreSQL提供数据加密功能，SSL链路加密和云盘加密 RDS PostgreSQL提供全加密数据库功能，选择SGX加密型
5 PostgreSQL 12.0 Primary 不支持
6 SQLServer 2016_std_ha Primary 不支持 实例类型为SQL Server企业版的主实例（只读实例不支持TDE）。只有阿里云主账号（即非子账号）登录才可以查看和修改TDE配置。开通TDE前需要先开通KMS。如果您未开通KMS，可在开通TDE过程中根据引导开通KMS。 版本支持明细见：https://help.aliyun.com/document_detail/145468.html
7 SQLServer 2016_ent_ha Primary 支持
8 SQLServer 2008r2 Primary 支持
9 PolarDB for MySQL 5.6 不支持 前提条件：集群版本为PolarDB MySQL 8.0PolarDB for Oracle 支持TDE：https://help.aliyun.com/document_detail/164154.html
10 PolarDB for Oracle 11 支持
